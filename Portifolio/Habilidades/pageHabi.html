My skills:

HTML, ou HyperText Markup Language, é a linguagem padrão utilizada para criar e estruturar conteúdo na web. É a espinha dorsal de qualquer página da web, fornecendo a estrutura básica para apresentar informações de forma visualmente atraente e acessível.
CSS, ou Cascading Style Sheets, é a linguagem utilizada para estilizar a apresentação de documentos HTML. Ele define a aparência e o layout de uma página da web, permitindo controlar cores, fontes, espaçamento e posicionamento dos elementos para criar uma experiência visualmente atraente e coesa para os usuários.
JavaScript é uma linguagem de programação de alto nível, leve e interpretada, amplamente utilizada para tornar páginas da web interativas e dinâmicas. Com sua sintaxe simples e versátil, pode ser incorporado em documentos HTML para manipular o conteúdo, responder a eventos do usuário e criar experiências interativas e responsivas.
Java é uma linguagem de programação orientada a objetos, robusta e versátil, conhecida por sua portabilidade e segurança. Amplamente utilizada no desenvolvimento de aplicativos de desktop, web e móveis, assim como em sistemas embarcados e servidores, Java oferece um ambiente de execução confiável e eficiente para uma variedade de aplicações.
